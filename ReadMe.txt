op mac:

1) installeer en gebruik virtual box 
https://www.virtualbox.org/wiki/Downloads

2) gebruik de debian virtual machine (zie presentatie), deze heeft monodevelop all geinstalleerd
3) open monodevelop
4) open de solution in ~/git/gameremix
5) unload het GameRemix.csproj project als die dat nog niet was
6) nu kun je GameRemixForLinux.csproj builden & runnen

op linux:
1) installeer mono 

https://www.mono-project.com/download/stable/
2) clone of fork gameremix vanuit git/gameremix
3) open monodevelop
4) open de solution in ~/git/gameremix
5) unload het GameRemix.csproj project als die dat nog niet was
6) nu kun je GameRemixForLinux.csproj builden & runnen
